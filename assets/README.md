# assets/ — art il·lustrat dels castellers

Posa aquí els PNG il·lustrats. El joc els carrega automàticament i, si en falta
cap, dibuixa la figura per codi (mai es queda en blanc).

## Fitxers (noms exactes)

| Fitxer | Peça |
|--------|------|
| `casteller_tronc.png` | casteller del tronc, de cara, braços estesos (la de prova) |
| `casteller_tronc_esquena.png` | el mateix, vist de darrere (sense cara) |
| `casteller_pinya.png` | persona de la pinya (mig cos, de dalt/darrere, mirant avall) |
| `casteller_acotxador.png` | nen acotxat, casc negre |
| `casteller_enxaneta.png` | nen dret, una mà amunt (aleta), casc negre |

## Regla de color (important)

Genera totes les peces amb **la camisa i el mocador de color VERMELL viu**
(saturat). El joc **recolora aquest vermell** al color de la colla de cada
jugador. La resta (pell, **pantalons blancs**, **faixa negra**, espardenyes,
**casc negre**) es manté igual.

- Fons **transparent**, figura **centrada**, **peus a baix de tot**.
- Mida ~512×1024 px (figures dretes).

## Com pujar els fitxers al repo (GitHub web)

1. Ves a **github.com/francescferremagrinya-coder/castells**.
2. Entra a la carpeta **`assets`**.
3. Botó **Add file → Upload files**.
4. Arrossega el PNG, posa-li el nom de la taula i **Commit changes**.

Quan hi sigui, avisa'm i l'integro (recolor + apilament + animació).
